# jit cmd note:

$ git reset --hard HEAD       (going back to HEAD)
$ git reset --hard HEAD^      (going back to the commit before HEAD)
$ git reset --hard HEAD~1     (equivalent to "^")
$ git reset --hard HEAD~2     (going back two commits before HEAD)
